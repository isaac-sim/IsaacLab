isaaclab_contrib.actuators#

Sub-package for thruster actuator models.

This package provides actuator models specifically designed for multirotor thrusters. The thruster actuator simulates realistic motor/propeller dynamics including asymmetric rise and fall time constants, thrust limits, and dynamic response characteristics.

Low-level motor/thruster dynamics with separate rise/fall time constants.

Integration scheme is Euler or RK4. All internal buffers are shaped (num_envs, num_motors). Units: thrust [N], rates [N/s], time [s].
